#  YourBento34 — Портфолио кондитера

> Персональный сайт кондитера с портфолио работ, каталогом продукции и формой обратной связи.

##  О проекте

**YourBento34** — это современный портфолио-сайт для частного кондитера. Посетители могут ознакомиться с готовыми работами, выбрать нужную категорию продукции и оставить заявку на заказ удобным способом.

##  Что на сайте

###  Портфолио выполненных проектов
- 20+ готовых десертов с фото «до/после»
- Свадебные торты, детские капкейки, корпоративные наборы
- Описания ингредиентов и сложности каждого заказа

### 🛒 Каталог продукции
- Торты на заказ (в т.ч. свадебные от 3 кг)
- Бенто-торты
- Корпоративные наборы
- Капкейки и эскимо
- Индивидуальная упаковка

### Форма обратной связи
Клиент выбирает интересующую позицию и связывается с кондитером через удобный канал для обсуждения деталей заказа.
